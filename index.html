Specificity

Flat file structure

// main.scss

@import 'settings/*';
@import 'tools/*';
@import 'generic/*';
@import 'elements/*';
@import 'objects/*';
@import 'components/*';
@import 'trumps/*';
1. Settings

Used with preprocessors

Fonts definitions, colors, variables, etc.

settings/_settings.colors.scss
$color-primary: #ce239a;
$color-secondary: #a7843b;

...
2. Tools

Globally used mixins and function

tools/_tools.clearfix.scss
@mixin clearfix() {

    &::after {
        content: '';
        display: table;
        clear: both;
        }
    }
3. Generic

Normalizing styles

Box-sizing

generic/_generic.box-sizing.scss
*,
*::before,
*::after {
    box-sizing: border-box;
    }
4. Elements

Bare HTML elements

Reset lists, anchors, etc.

elements/_elements.headings.scss
h1 {
    @include font-size($font-size-h1);
    }

h2 {
    @include font-size($font-size-h2);
    }

h3 {
    @include font-size($font-size-h3);
    }

...
5. Objects

Simple & reusable

Containers, grids, global animations, etc.

objects/_objects.box.scss
.o-box {
    display: block;
    padding: $global-spacing-unit;

    /* Size variants
    ================ */

    &--flush {
        padding: 0;
        }

    &--small {
        padding: $global-spacing-unit-tiny;
        }

    &--large {
        padding: $global-spacing-unit-large;
        }
    }
